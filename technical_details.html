memopay technical information



[1] memopay messages

memopay sends UTF-8 encoded messages to P2PKH addresses via OP_RETURN. In each message memopay uses it's official Prefix: 0x6d656d6f706179

Message example:


Encoded: OP_RETURN 6d656d6f706179 20596f7572206d657373616765206865726520
Decoded: memopay Your message here
